Comment ça marche?

1. Créez une clé API gratuite sur Google AI Studio. Vous bénéficiez de 60 requêtes/minute gratuitement.

2. Importez votre CV sur la page Profil pour activer le matching intelligent.

3. Analysez les offres - L'IA comparera votre profil avec chaque offre et vous donnera un score de compatibilité.

4. Générez des lettres - L'IA peut vous aider à rédiger des lettres de motivation personnalisées.

Confidentialité: Votre CV n'est jamais envoyé en entier à Gemini. Seules les informations pertinentes pour le matching sont transmises.
